## How to basic

- Run with visualizer

```bash
make tests ARGS="5 800 200 200 10"
```

- Run normal

```bash
make run ARGS="5 800 200 200 10"
```

- Run valgrind

```bash
make runv ARGS="5 800 200 200 10"
```
